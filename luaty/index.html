Luaty language

Luaty is an indent sensitive language with partial type inference that compiles to Lua.

Overview

Luaty stands for Lua with more or less typing. 'Typing' has dual meaning here, and summarizes the benefit of using Luaty.

Type inference

Luaty type checker statically infers variable types in the code and warns about their inconsistent usage. Although it is not possible to correctly and fully infer all variable types, some warnings are quite helpful in discovering issues in the code.

Type less

Due to offside syntax, Luaty could use less syntax boilerplate than Lua:

  • no more end
  • no more do after for and while
  • no more then after if

To sum it up: we type check, you type less
